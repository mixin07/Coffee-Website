### **Key Features**

* **Dynamic Hero Section:** A welcoming entrance with clear Call-to-Action (CTA) buttons for ordering or contacting the shop.
* **About Us Section:** Details the shop's mission and provides direct social links to the developer's LinkedIn and GitHub profiles.
* **Digital Menu:** A structured display of hot/cold beverages, refreshments, special combos, and desserts.
* **Testimonial Slider:** An interactive carousel built with **Swiper.js** that highlights verified customer feedback.
* **Photo Gallery:** A clean grid layout showcasing the coffee shop's environment and products.
* **Contact & Location Hub:** Provides essential business information, including an address, email, phone number, and operating hours.
